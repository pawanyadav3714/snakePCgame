# 🐍 Snake Game (Mobile + Desktop)

A classic **Snake Game** built using **HTML, CSS, and JavaScript**, designed to work on both **desktop (keyboard controls)** and **mobile (touch controls)**.

---

## 🚀 Features

* 🎮 Smooth snake movement with increasing difficulty
* 🍎 Random food generation
* 🔊 Sound effects (food, movement, game over, background music)
* 📱 Mobile-friendly touch controls
* 💾 High score stored using Local Storage
* 🎨 Responsive UI with grid-based board

---

## 🛠️ Tech Stack

* **HTML5** → Structure of the game 
* **CSS3** → Styling and layout 
* **JavaScript (Vanilla)** → Game logic and interactions 

---

## 🎮 How to Play

### 🖥️ Desktop Controls

* ⬆️ Arrow Up → Move Up
* ⬇️ Arrow Down → Move Down
* ⬅️ Arrow Left → Move Left
* ➡️ Arrow Right → Move Right

### 📱 Mobile Controls

* Use on-screen buttons:

  * **U** → Up
  * **D** → Down
  * **L** → Left
  * **R** → Right

## 💡 Game Logic Overview

* The snake moves using a **grid system (70x70)**
* Food appears randomly on the board
* Score increases when food is eaten
* Game ends when:

  * Snake hits the wall
  * Snake collides with itself
* High score is saved using **localStorage**
